# DoubleSave Australian retailer discount-code index

A machine-readable index of the Australian retailer pages currently covered by [DoubleSave](https://www.doublesave.com/), an AU-first discount-code and shopping-savings reference.

## What is included

- Retailer name and slug
- Shopping category
- Canonical DoubleSave retailer-page URL
- Public coverage only — no affiliate credentials, private research notes, or unpublished offer data

Current coverage: **389 Australian retailers**.

Browse the full consumer directory: [DoubleSave Australian discount codes](https://www.doublesave.com/au/discount-codes/).

## Data files

- [`data/retailers.json`](data/retailers.json)
- [`data/retailers.csv`](data/retailers.csv)

## Methodology

The index is generated from DoubleSave's public retailer-page inventory. Individual pages distinguish reported discount-code leads from stronger source or checkout evidence and include practical constraints such as delivery thresholds, memberships, exclusions, and stacking rules where available.

Examples:

- [Amazon Australia discount codes](https://www.doublesave.com/au/discount-codes/amazon/)
- [Kmart Australia discount codes](https://www.doublesave.com/au/discount-codes/kmart/)
- [Jetstar Australia discount codes](https://www.doublesave.com/au/discount-codes/jetstar/)
